Exercise 1: Write, and test JavaScript scripts for the problems that follow. When required to write a function, you must include a script to test the function with at least two different data sets. In all cases, for testing, you must write an XHTML file that references the JavaScript file.

1.1 → Output: A table of the numbers from 5 to 15 and their squares and cubes using alert

1.2 → Input: A number n, using prompt

Output: The first n Fibonacci numbers, which are defined as in the sequence 1, 1, 2, 3, ...

Where each number in the sequence after the second is the sum of the two previous numbers. You must use document.write to produce the output

1.3 → Input: Three numbers, using prompt to get each

Output: The largest of the three input numbers.

1.4 → Input: A text string, using prompt

Output: Either “Valid name” or “Invalid name”, depending on whether the input names fit the required format, which is Last Name, First Name, Middle initial

Where neither of the names can have more than 15 characters.

1.5 → Input: A text string, using prompt

Output: Either “Valid Phone number” or “Invalid phone number”, depending on whether the input phone numbers fit the required format, which is (###)-###-#### or ###-###-####

Where # is any single digit in the range 0-9.

1.6 → Input: A line of text, using prompt

Output: The words of the input text, in alphabetical order.

1.7 → Function: first_vowel

Parameter: A String

Returns: The posistion in the string of the leftmost vowel.
